## 2) How it connects to me (the right architecture)

### Strong take
**Do not connect the phone directly to OpenClaw with the master gateway token.**  
Use a thin **Mobile API relay** on your Mac as a security/control layer.

### Architecture
- **iPhone app (React Native/Expo or SwiftUI)**
  - UI only
  - Auth with passkey/device auth
- **Mobile API relay (Node/Fastify)**
  - Stores gateway token securely on host
  - Owns OpenClaw connections
  - Enforces allowlist and rate limits
- **OpenClaw Gateway**
  - WS + HTTP on same port (default `18789`)
  - Source of truth for chat/events/tools

### OpenClaw interfaces to use
1. **Primary chat path: WebSocket**
   - Methods: `chat.send`, `chat.history`, `chat.abort`, `chat.inject`
   - Events: `chat`, `agent`, `presence`, `tick`, `health`
   - Best for live mobile chat and deterministic session behavior

2. **Operational actions: HTTP tools invoke**
   - `POST /tools/invoke`
   - Use for controlled single-tool calls powering dashboards/actions

3. **Optional compatibility path**
   - `POST /v1/chat/completions` (disabled by default, enable in config if needed)

### Remote access
- Use **Tailscale** or **SSH tunnel** to reach gateway safely
- Keep gateway bind local/loopback unless you intentionally harden remote exposure

## 3) Security model (non-negotiable)

- Phone never gets raw gateway master token
- Relay issues short-lived app JWT (15-30 min)
- Relay maps user role -> allowed actions
- Tool allowlist only, deny everything else by default
- Audit log every action with requestId + actor + result
- Encrypted local storage on phone, biometric unlock
- Optional “high-risk action requires re-auth” toggle

---

## 4) Data model (minimal and effective)

- `sessions` (id, title, lastMessageAt)
- `messages` (sessionId, role, content, status, timestamps)
- `events` (severity, source, payload, createdAt)
- `alerts` (P1/P2/P3, state, acknowledgedBy)
- `approvals` (action, riskTier, status, expiresAt)
- `dash_metrics` (name, value, window, capturedAt)

Keep storage local-first on phone with sync from relay.
